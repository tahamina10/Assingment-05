1️ What is the difference between var, let, and const?
Ans: var → It is function-scoped and can be redeclared and updated.

let → It is block-scoped. It can be updated but cannot be redeclared in the same scope.

const → It is also block-scoped. It cannot be updated or redeclared after declaration.

2️ What is the spread operator (...)?
Ans: The spread operator (...) is used to expand elements of an array or properties of an object. It is commonly used to copy arrays/objects or to merge them.

3 What is the difference between map(), filter(), and forEach()?
 Ans: map() → Executes a function on every array element and returns a new array.

filter() → Returns a new array containing elements that match a given condition.

forEach() → Executes a function on each array element but does not return a new array.

4️ What is an arrow function?
 Ans: An arrow function is a shorter way to write a function in JavaScript using the => syntax. It makes the code more concise and readable.

5️ What are template literals?
Ans:  Template literals are a modern way to create strings in JavaScript using backticks ` `. They allow embedding variables or expressions inside a string using ${}.
